# Skills

Hermes Agent skills for using the brainstorm tool.

## Install

Copy the skill file to your Hermes skills directory:

```bash
mkdir -p ~/.hermes/skills/productivity/
cp skills/brainstorm-cli.md ~/.hermes/skills/productivity/brainstorm-cli/SKILL.md
```

Or symlink:

```bash
ln -s $(pwd)/skills/brainstorm-cli.md ~/.hermes/skills/productivity/brainstorm-cli/SKILL.md
```

## Available skills

- **brainstorm-cli** — Multi-AI brainstorm CLI usage guide, strategy recommendations, command reference
